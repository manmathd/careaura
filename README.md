## Features
- Demo OTP login (client-side; for demo only)
- Local reminders (stored in `localStorage`)
- SOS button with caregiver alert log
- Simulated health statistics (heart rate, steps, sleep, fall detection)
- Caregiver dashboard and local messages
- Single-page app (no page reloads)
- Optional sound for reminders (toggle in Settings)
- Accessibility improvements (focus management & aria attributes)
